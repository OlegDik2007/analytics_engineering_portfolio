# KPI dictionary (basic)

- **DAU (visitors):** distinct visitor_id per day
- **Sessions:** distinct session_id per day
- **Top Pages:** count events grouped by page_url
- **Funnel:** session-level steps
  - page_view → view_details → contact_agent
- **Geo Mix:** events/visitors grouped by country/state/city

Not finished list, add more as you build.
